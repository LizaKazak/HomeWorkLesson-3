# Синтаксис языка Markdown

1. Для того, чтобы выделить текст жирным, необходимо с обеих сторон выделить слово или фразу двумя звёздочками. 

**жирный текст**

*курсив*

Ненумерованный список

* элемент 1;

* элемент 2;

* элемент 3.

The end

Добавим цитату

> это - цитата

>> это - цитата в цитате
